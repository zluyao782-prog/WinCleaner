# WinCleaner Project Cards

## P0

- Windows 实机回归矩阵
  - 覆盖 Win10/11、UAC、服务、注册表、回收站、Explorer、WMI

- 高风险操作报告
  - 为清理、终止进程、更新控制输出统一结果摘要和失败明细

- 安全逻辑测试扩展
  - 覆盖更新控制、日志过滤、清理统计、进程保护和结果勾选

- 统一后台任务框架
  - 统一进度、异常、取消、超时和日志接入

## P1

- 配置继续外置
  - 把阈值、默认路径、日志级别、危险开关改成配置项

- 日志面板增强
  - 增加来源过滤、关键字搜索和完整诊断导出

- 持久化清理排除规则
  - 支持按路径、目录、扩展名保存排除项

- 统一版本来源
  - 让应用、tag、Release、文档使用同一版本号来源

## P2

- 拆分文档体系
  - 用户手册、开发手册、发布手册、故障排查手册

- 规则插件化设计
  - 设计清理规则和保护规则的可扩展加载机制
