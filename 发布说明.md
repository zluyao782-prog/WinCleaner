# WinCleaner v1.0 发布说明

## 📦 发布包内容

### 核心文件
- `WinCleaner.exe` - 主程序（需要在Windows环境下构建）
- `README.md` - 详细使用说明
- `项目概览.md` - 项目开发说明

### 源代码结构
```
WinCleaner/
├── main.py                     # 程序入口
├── core/                       # 核心功能模块
│   ├── engine.py              # 统一调度引擎
│   ├── sysinfo.py             # 系统信息获取
│   ├── update_ctrl.py         # Windows更新控制
│   ├── disk.py                # 磁盘分析功能
│   ├── cleaner.py             # 垃圾文件清理
│   └── process_mgr.py         # 进程管理
├── ui/                        # 用户界面
│   ├── main_window.py         # 主窗口框架
│   └── pages/                 # 功能页面
├── config/                    # 配置文件
├── assets/                    # 资源文件
├── requirements.txt           # Python依赖
├── build.bat                  # Windows构建脚本
├── build.sh                   # Unix构建脚本
└── version_info.txt           # 版本信息
```

## 🚀 构建说明

### Windows环境构建
1. 确保已安装Python 3.10+
2. 以管理员身份运行命令提示符
3. 执行构建脚本：
   ```cmd
   build.bat
   ```
4. 构建完成后在`dist`目录找到`WinCleaner.exe`

### 构建要求
- **操作系统**: Windows 10/11 (64位)
- **Python版本**: 3.10或更高
- **管理员权限**: 推荐（避免权限问题）
- **网络连接**: 需要下载依赖包

## 🔧 安装说明

### 用户安装步骤
1. 下载`WinCleaner.exe`
2. 右键选择"以管理员身份运行"
3. 在UAC提示中点击"是"
4. 程序自动启动

### 系统要求
- **操作系统**: Windows 10 (1903+) / Windows 11
- **架构**: 64位系统
- **权限**: 管理员权限
- **磁盘空间**: 至少50MB可用空间

## ⚠️ 重要提示

### 安全警告
1. **管理员权限**: 程序需要管理员权限才能正常工作
2. **数据备份**: 使用前建议备份重要数据
3. **更新风险**: 禁用Windows更新可能带来安全风险
4. **进程终止**: 终止进程可能导致数据丢失

### 兼容性说明
- ✅ Windows 10 (1903及以上版本)
- ✅ Windows 11 (所有版本)
- ❌ Windows 7/8/8.1 (不支持)
- ❌ 32位系统 (不支持)

## 🐛 已知问题

### 当前版本限制
1. **语言支持**: 仅支持中文界面
2. **主题支持**: 仅支持浅色主题
3. **网络功能**: 无在线更新功能
4. **日志管理**: 日志文件需手动清理

### 解决方案
- 大部分问题将在后续版本中修复
- 如遇问题请查看日志文件：`logs/wincleaner_YYYYMMDD.log`

## 📞 技术支持

### 故障排除
1. **程序无法启动**: 检查是否以管理员权限运行
2. **功能异常**: 查看日志文件获取详细错误信息
3. **权限问题**: 确保在Windows 10/11系统上运行
4. **性能问题**: 关闭其他占用资源的程序

### 联系方式
- 查看`README.md`获取详细使用说明
- 检查`logs`目录下的日志文件
- 确保系统满足最低要求
