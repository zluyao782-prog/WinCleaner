# WinCleaner - Windows 系统清理工具

> 🧹 专业的Windows系统清理和优化工具，让您的电脑重获新生

WinCleaner 是一个功能强大的 Windows 系统清理和优化工具，帮助用户轻松管理系统资源、清理垃圾文件、控制系统更新和监控进程。

## ✨ 主要功能

### 🖥️ 系统信息
- 📊 实时显示操作系统版本、CPU、内存等硬件信息
- ⏱️ 监控系统运行时间和性能状态
- 🔄 查看 Windows 更新状态

### 🔄 更新控制
- 🛑 一键禁用/启用 Windows 自动更新
- 🔒 通过注册表策略和服务控制双重保障
- 📋 实时显示相关服务状态

### 💾 磁盘分析
- 📈 显示所有磁盘分区的使用情况
- 🔍 扫描指定目录下的大文件
- 📊 可视化磁盘空间分布

### 🗑️ 垃圾清理
- 🧹 支持多种垃圾文件类别：
  - 系统临时文件
  - 微信/QQ 缓存文件
  - 浏览器缓存
  - 缩略图缓存
  - 系统日志文件
- 👀 先扫描预览，确认后清理
- 🗂️ 支持清空回收站

### ⚙️ 进程管理
- 🔍 实时监控所有运行进程
- 🚨 智能标记高负荷进程（CPU/内存占用高）
- 🔎 支持搜索和排序功能
- ⚡ 一键终止高负荷进程
- 🛡️ 系统关键进程保护机制

## 💻 系统要求

| 项目 | 要求 |
|------|------|
| **操作系统** | Windows 10 (1903+) / Windows 11，64位 |
| **权限要求** | 需要管理员权限运行 |
| **磁盘空间** | 至少 50MB 可用空间 |
| **运行时** | 无需安装额外软件 |

## 🚀 快速开始

### 📥 下载安装
1. 下载 `WinCleaner.exe`
2. 右键选择"以管理员身份运行"
3. 在UAC提示中点击"是"
4. 程序自动启动

### 🎯 首次使用
1. **系统信息**: 启动后自动加载，可点击"刷新信息"更新
2. **更新控制**: ⚠️ 谨慎使用，禁用更新可能带来安全风险
3. **磁盘分析**: 选择路径后点击"开始扫描"
4. **垃圾清理**: 选择类别→扫描→确认→清理
5. **进程管理**: 实时监控，可搜索、排序、终止进程

## ⚠️ 安全提示

| 警告类型 | 说明 |
|----------|------|
| 🔒 **更新风险** | 禁用Windows更新会阻止安全补丁安装 |
| ⚡ **进程风险** | 终止进程可能导致程序异常或数据丢失 |
| 🗑️ **清理风险** | 清理文件前请确认不再需要 |
| 🛡️ **系统保护** | 系统关键进程受保护，无法终止 |

## 🔧 故障排除

### 常见问题

**Q: 程序无法启动？**  
A: 确保以管理员权限运行，检查系统是否为Windows 10/11

**Q: 某些功能不可用？**  
A: 检查是否具有管理员权限，查看日志文件获取详细信息

**Q: 清理后系统异常？**  
A: 部分清理操作不可逆，建议使用前备份重要数据

**Q: 进程无法终止？**  
A: 系统关键进程受保护，或进程已自动退出

### 📋 日志文件
程序运行日志保存在 `logs` 目录下，文件名格式：`wincleaner_YYYYMMDD.log`
