# 资源文件说明

## 图标文件

当前版本暂时不包含自定义图标，使用系统默认图标。

如需添加自定义图标：
1. 将 `.ico` 格式的图标文件放置在此目录
2. 命名为 `app.ico`
3. 在构建脚本中取消注释 `--icon` 参数

## 图标要求

- 格式：ICO
- 推荐尺寸：16x16, 32x32, 48x48, 256x256像素
- 支持透明背景

## 在线图标转换工具

- [ICO Convert](https://icoconvert.com/)
- [Favicon Generator](https://www.favicon-generator.org/)
- [ConvertICO](https://convertico.com/)
